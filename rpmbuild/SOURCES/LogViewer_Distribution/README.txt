QUICK START:
1. Double-click LogViewer.exe to run the application
2. Click "Open Log File" and select test.log to try it out
3. Search for "ERROR" or "INFO" to see text highlighting

WHAT'S INCLUDED:
- LogViewer.exe          - Main application (36MB)
- config.yml             - Sample configuration file
- test.log               - Sample log file for testing
- Launch_LogViewer.bat   - Alternative launcher
- README_Windows.md      - Detailed documentation
- INSTALLATION_GUIDE.md  - Complete setup guide
- README.txt             - This file

SYSTEM REQUIREMENTS:
- Windows 10 or later (64-bit)
- 100MB RAM minimum
- 50MB free disk space
- No Python installation required!

FEATURES:
- Fast search with highlighting
- Supports .log, .out, .txt files
- Configurable color highlighting
- Dark theme interface
- Handles large files (multi-GB)
- Portable - no installation needed

USAGE:
- Open files: Drag & drop or use "Open Log File" button
- Search: Type in search box and press Enter
- Navigate: Use F3 (next) and Shift+F3 (previous)
- Configure: Click "Configure Highlighting" for custom colors

KEYBOARD SHORTCUTS:
- Ctrl+O: Open file
- Ctrl+F: Focus search
- F3: Find next
- Shift+F3: Find previous
- Escape: Clear search
- Ctrl+Q: Quit
